General Instruction:

1. Copy the entire Resequencer_DB_Script folder for Code deployment.
2. Do not remove any file from the folder.
3. Execute the master.sql file for Resequencer DB setup.
4. A log file,master_script_log_v1.0.log,  will be generated after master.sql file execution.
5. Verify the log file, ensure there is no error in DB Objects setup.
6. In case of any error, you may revert all the changes made, by executing Rollback_master.sql file, under Rollback folder.
7. It will also generate a log file; Rollback_master_script_log_v1.0.log
